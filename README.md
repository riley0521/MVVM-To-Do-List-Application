# Android Application about "To-Dos"

Basic To-Do application with application of clean architecture (without use-case).

### Tech Stack:
- Navigation Architecture
- Dagger Hilt
- Room Database
- Kotlin Coroutines and Flow
- Lifecycle - LiveData and ViewModel
